# Design fundamentals: Teacher cards

På den vedlgte nettsiden er det lagt inn data om lærerne på 2IT. Nå er det ingen CSS lagt inn på nettsiden, så det ser ikke bra ut. Du skal skrive CSS for å presentere lærerne på hvert sitt kort. Legg også inn nødvendige klasser i HTMLen og ekstra HTM-tagger om nødvendig.

Husk designprinsippene:
- Whitespace
- Alignment
- Contrast
- Scale
- Typography
- Color
- Visual Hierarchy

# Instruksjoner for kloning av nettsiden

Denne nettsiden bruker rammeverket Vite Vanilla. For å få nettsiden til å kjøre, følg disse stegene:

1. Klon nettsiden til Visual Studio Code
2. I terminalen, kjør følgende kommandoer:

```shell
npm install
npm run dev
```
